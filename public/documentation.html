letFlix App API Endpoints Documentation

the API serves as a platform for managing user profiles and their movie preferences, along with providing access to a database of movies, searchable by various criteria like title, genre, and director. This makes it a useful tool for applications related to movie recommendations, user preference tracking, and film database management.

Operation Method URL Request body data format Response body data format
Create User POST /users A JSON object holding user data, structured like this. Example: { "name": "Alice", "favoriteMovies": ["Inception", "Interstellar"] } A JSON object holding user data plus an ID and an array for placing favorite movies. Example: { "id": "generated-uuid", "name": "Alice", "favoriteMovies": ["Inception", "Interstellar"] }
Update User PUT /users/:id A JSON object holding user data, structured like this. Example: { "name": "Alice Updated" } A JSON object holding user data plus an ID and an array for placing favorite movies. Example: { "id": "existing-user-id", "name": "Alice Updated", "favoriteMovies": ["Inception", "Interstellar"] }
Add Favorite Movie POST /users/:id/:movieTitle NONE A JSON object which adds a movie title to a user's list of favorite movies. Example: Response: Titanic has been added to user 1's array
Remove Favorite Movie DELETE /users/:id/:movieTitle NONE A JSON object which removes a movie title to a user's list of favorite movies. Example: Response: Titanic has been removed from user 1's favorites.
Delete User DELETE /users/:id NONE A JSON object which removes a User. Example: Response: user 1 has been removed.
Get All Movies GET /movies NONE A JSON object holding an array with all movies.
Get Movie by Title GET /movies/:title NONE A JSON object holding user data plus an ID and an array for placing favorite movies. Example: { "title": "Titanic", "genre": { "name": "Drama, Romance", "description": "An epic romance set against the ill-fated maiden voyage of the R.M.S. Titanic." }, "director": { "name": "James Cameron", "Bio": "A filmmaker known for his ambitious and visually spectacular films, often pushing the boundaries of film technology.", "Birth": "August 16, 1954, in Kapuskasing, Ontario, Canada" }, "image URL": "https://www.imdb.com/title/tt0120338/mediaviewer/rm10105600/?ref_=tt_ov_i", "feature": false }
Get Movies by Genre GET /movies/genre/:genreName NONE A JSON object holding data about a genre containing name and description properties. Example: { "name": "Fantasy", "description": "In film and television, the fantasy genre encompasses fiction that involves supernatural elements and settings beyond the realm of reality. It often includes magical creatures, mystical forces, and worlds filled with wonder and adventure. Fantasy allows for the exploration of themes like heroism, morality, and the struggle between good and evil in a setting detached from mundane reality." }
Get Movies by Director GET /movies/directors/:directorName NONE A JSON object holding data about a director. Example: { "name": "Steven Spielberg", "Bio": "A highly acclaimed director known for his impactful storytelling and versatility in various film genres.", "Birth": "December 18, 1946, in Cincinnati, Ohio, USA" }
