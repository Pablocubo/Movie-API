letFlix App API Endpoints Documentation

the API serves as a platform for managing user profiles and their movie preferences, along with providing access to a database of movies, searchable by various criteria like title, genre, and director. This makes it a useful tool for applications related to movie recommendations, user preference tracking, and film database management.

Operation Method URL Description
Create User POST /users Adds a new user.
Update User PUT /users/:id Updates a user's information.
Add Favorite Movie POST /users/:id/:movieTitle Adds a movie to user's favorites.
Remove Favorite Movie DELETE /users/:id/:movieTitle Removes a movie from user's favorites.
Delete User DELETE /users/:id Deletes a user.
Get All Movies GET /movies Retrieves all movies.
Get Movie by Title GET /movies/:title Retrieves a specific movie by title.
Get Movies by Genre GET /movies/genre/:genreName Retrieves movies of a specific genre.
Get Movies by Director GET /movies/directors/:directorName Retrieves movies by a specific director.
